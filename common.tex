% \quad Всё возрастающая конкуренция на рынке подталкивает современные приборостроительные
% предприятия к постоянному улучшению и развитию производства. В настоящее
% время одними из самых перспективных способов достижения высокой конкурентоспособности
% являются совершенствование используемых технологий и повышение эффективности
% технологической подготовки производства (ТПП) за счёт применения современных
% средств автоматизации.
% % Оба этих аспекта тесно взаимосвязаны.
% % Внедрение новых передовых технологий существенно повышает требования к разрабатываемому
% % технологическому процессу и, как следствие, стимулирует приборостроительные
% % предприятия к внедрению всё новых и новых средств автоматизации ТПП, призванных
% % сократить время, затрачиваемое на разработку технологических процессов, повысить
% % их качество и, в конечном итоге, увеличить прибыль за счёт ускорения вывода 
% % продукции предприятия на рынок.
% Особенно это актуально для предприятий, использующих
% передовые технологии, требующие дополнительных инженерных изысканий, например,
% производство изделий из полимерных композиционных материалов (ПКМ).
\textls[20]{Всё возрастающая конкуренция на рынке подталкивает современные приборостроительные
предприятия к~пос\-тоянному улучшению и развитию производства. В настоящее
время одним из наиболее перспективных способов достижения высокой конкурентоспособности
является повышение эффективности
технологической подготовки производства (ТПП) за счёт применения современных
средств автоматизации.
Особенно это актуально для предприятий, использующих
передовые технические решения и технологии, требующие дополнительных инженерных изысканий.}

Большой вклад в разработку базовых принципов построения и взаимодействия автоматизированных систем
технологической подготовки произ\-водства (АСТПП) внесли С.\,П.~Митрофанов, В.\,И.~Аверченков,
Г.\,К.~Горанский, В.\,Д.~Цветков, Н.\,М.~Капустин, В.\,В.~Павлов, В.\,М.~Вальков,
А.\,Н.~Филиппов, Д.\,Д.~Куликов, Б.\,С.~Падун, Е.\,И.~Яблочников
и многие другие.
% Существует стандарт (ГОСТ Р ИСО 10303-21--2002), регламентирующий
Тем не менее, в~условиях современного наукоёмкого производства
многие из этих принципов нарушаются. В первую очередь это связано с
отсутствием универсальной интеграционной среды, способной собрать воедино
различные инструментальные средства автоматизации ТПП.


% Как известно, основной целью ТПП является оптимальное по срокам и ресурсам обеспечение
% технологической готовности производства к изготовлению изделий в соответствии с
% требованиями заказчика или рынка данного класса изделий~\cite{Matalin-2008}. Задействование средств
% автоматизации должно сокращать время этого процесса. Но здесь возникает очевидная
% проблема~--- увеличение номенклатуры выпускаемой продукции во всех типах производства
% и существенное усложнение технологического процесса вследствие применения 
% новых технологий ведёт к необходимости использования сразу нескольких программных
% средств автоматизации ТПП, причём использования их в комплексе.
% 
% Данное положение усугубляется тем, что автоматизированные системы технологической
% подготовки производства представляют собой очень широкий класс программного обеспечения.
% Из этого следует, что ни один производитель подобных программных средств не в состоянии
% создать единую линейку продуктов, охватывающую весь комплекс мероприятий,
% относящихся к ТПП. Каждое предприятие вынуждено создавать свою собственную, уникальную
% информационную среду технологической подготовки производства, зачастую состоящую из
% \emph{слабо структурированного} набора самых разнообразных программных средств автоматизации.
% Попытки привлечь специалистов по интеграции в основном не дают желаемого эффекта.
% Это обусловлено тем, что организации, оказывающие подобные услуги не обладают
% всей полнотой знаний о производственном процессе каждого конкретного предприятия,
% следовательно вынуждены предлагать некоторые универсальные решения, часто не
% улучшающие процесс автоматизации ТПП, а, наоборот, ухудшающие его.
% 
% Второй проблемой, с которой неизбежно сталкивается любое предприятие, вставшее на путь
% завоевания новых для себя потребительских рынков и желающее создавать продукцию,
% востребованную не только в России и странах СНГ, но и за рубежом является
% необходимость работать по двойным стандартам. Данное положение ещё сильнее снижает
% внутрипроизводственную интеграцию вследствие необходимости постоянной синхронизации
% одних и тех же технологических данных между разными автоматизированными системами.
% Опыт показывает, что данная задача может быть решена \emph{только} специалистами
% предприятия, ведь они как никто другой знают все тонкости работы своей организации.
% 
% Всё вышесказанное приводит к тому, что информационно-управляющее окружение технологической
% подготовки большинства отечественных предприятий представляет собой некий гибрид различных
% автоматизированных систем (как отечественных, так и импортных), специализированных
% модулей, созданных сотрудниками самого предприятия и некоторого подобия единой
% интеграционной среды, основная функция которой сводится к поддержанию в актуальном
% состоянии разрозненных баз данных различных
% автоматизированных системам ТПП. При этом вся ответственность за работоспособность
% информационного окружения ложится на рядовых сотрудников предприятия, которые помимо своих
% основных обязанностей должны также проводить постоянный мониторинг данных и знаний, с которыми
% они работают и, в случае необходимости, вводить их во все используемые на предприятия
% системы. Очевидно, что такое подход даже хуже, чем использование классической
% <<бумажной технологии>>, к которой вынуждены возвращаться многие предприятия,
% не сумевшие создать адекватное структурированное информационное поле всех 
% используемых технологических систем.
% 
Решение данной проблемы наиболее целесообразно с применением методов распределённого
искусственного интеллекта, базовой дисциплиной которого является теория
многоагентных систем (МАС). Применение МАС для решения задач
технологической подготовки производства позволит создать открытую среду интеграции
технологических данных и~знаний, построенную на простой модели расширения функциональности
и~горизонтального масштабирования информационного пространства технологической подготовки \mbox{производства}.
{\parfillskip=0pt

}

\textls[25]{На сегодняшний день существует достаточное количество работ, пос\-вящённых
применению многоагентных систем в промышленности и~производстве, но~ни в одной из них не представлено
детальное исследование рассматриваемой предметной области~--- технологической
подготовки приборостроительного производства. На основе проводимого исследования необходимо
подготовить методику, применение которой даст возможность повысить уровень
автоматизации при~решении задач технологической подготовки и~увеличит
структуризацию информационной среды современного \mbox{предприятия}.}

% \textls[20]{Всё вышесказанное подтверждает актуальность проектирования и разработки многоагентной
% системы, являющейся единым интеграционным ядром автоматизированной среды
% технологической подготовки приборостроительного производства.}

Всё вышесказанное подтверждает актуальность проектирования и разработки многоагентной
системы, позволяющей повысить степень интеграции информационного пространства и
упростить решения задач технологической подготовки производства.
}

\mkcommonsect{objective}{Цель диссертационной работы}{%
% состоит в исследовании и разработке многоагентной системы,
% являющейся единым интеграционным ядром информационно-управляющей платформы для
% решения задач технологической подготовки производства.
состоит в совершенствовании методов автоматизации технологической подготовки
производства путём использования информационно-управляющей платформы,
созданной на базе многоагентной системы.

Для достижения поставленной цели в диссертационной работе потребовалось решить
следующие {\bfseries основные задачи}:
\begin{itemize}
\item Исследовать существующие методы построения распределённых одноранговых
    многоагентных систем, их архитектуры и области применения, а также инструментальные
    средства разработки. 
\item Разработать математические модели многоагентной среды и многоагентной системы
\item Описать язык представления технологических данных и~знаний, используемый агентами.
% \item {Исследовать и разработать базовые методы взаимодействия
%     агентов внутри многоагентной системы технологического назначения.\parfillskip=0pt}
% \item Проанализировать основные инструментальные средства проектирования и разработки
%     многоагентных систем.
\item Разработать систему моделирования информационного пространства технологической подготовки
    производства, базирующуюся на~\mbox{концепциях} <<облачных>> вычислений и виртуальных рабочих мест.
% \item Разработать и программно реализовать модель многоагентной интеллектуальной интеграции
%     автоматизированных систем технологической подготовки производства.
\item Реализовать многоагентную систему и опробовать её при решении конкретных технологических задач.
\end{itemize}
}
